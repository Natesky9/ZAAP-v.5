ZAAP v.5

ZAAP is a multiplayer game that focuses on technology, economy, combat, and production.
Inspired by several games and mods, such as Freelancer, Thermal Expansion, No Man's Sky, and Motherload, this game strives to provide multiple paths to the peak of technology, economy, and warfare.
